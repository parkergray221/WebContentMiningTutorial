Companies Using Content Mining

EBay

EBay is now using web mining techniques to analyze bidding behavior to determine if a bid is fraudulent. Recent efforts are geared towards understanding participants’ bidding behaviors/patterns to create a more efficient auction market.

Google Search

One of the latest services offered by Google is Google News. It integrates news from the online versions of all newspapers and organizes them categorically to make it easier for users to read “the most relevant news.” It seeks to provide latest information by constantly retrieving pages from news sites worldwide that are being updated on a regular basis, and Google performs this retrieval with web content mining.

Personalized Portal for the Web—MyYahoo

Mining MyYahoo usage logs provides Yahoo valuable information regarding a person's web usage habits, enabling Yahoo to provide personalized content, which has raised user engagement in Yahoo webpages.

V-TAG Web Mining Server-Connotate Technologies

This web mining server supports information agents that monitor, extract and summarize information from web sources. This server comes with a user-friendly GUI. Automation of tracking and summarizing helps businesses and enterprises to analyze the whole breadth of the Internet's webpages more easily.

Web-Wide Tracking

“Web-wide tracking” tracks an individual across all sites they visit and is predictably a controversial technology. It can provide an understanding of an individual’s lifestyle and habits on a scale that is not achievable by any one website proprietor, which is clearly an attractive point to marketers for commercial purposes.

Understanding Web Communities-AOL

One of the biggest successes for America Online has been its sizable and loyal customer base. A large portion of this customer base participates in various AOL communities, which are collections of users with similar interests. Over time these communities have grown to be well-visited waterholes for AOL users with shared interests. Applying web mining to the data collected from community interactions provides AOL with a very good understanding of its communities, which it has used for targeted marketing through advertisements and e-mail solicitation.
