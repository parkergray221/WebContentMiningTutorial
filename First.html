Web Content Mining

Group 1 (Raymond Muller, Vennela Vysyaraju, and Parker Gray)

Web content mining is a type of data mining that aims to extract useful information from the contents of webpages. This type of mining utilizes webscrapers which are used to comb through a webpage and find relevant data to whatever query the user might input. The purpose of this tutorial is to explain Web content mining in more detail and its useful applications in the world.
